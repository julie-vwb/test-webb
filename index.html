เกมอื่นๆสามารถสอบถามเพิ่มเติมได้คับ

📌 รายละเอียดการเติมเกม

1️⃣ เติมแบบ UID

  • เติมผ่านรหัสไอดี ไม่จำเป็นต้องเข้าเกม
  • ปลอดภัย ความเสี่ยงน้อย
  • ราคาจะสูงกว่าแบบ Mail-Pass เล็กน้อย

2️⃣ เติมแบบ ID - Pass (Mail - Pass)

  • ใช้ Username และ Password ในการเข้าเติม
  • ราคาถูกกว่า
  • ราคามีการเปลี่ยนแปลงบ่อย

บางเกมสามารถเติมได้ทั้ง 2 แบบ สอบถามก่อนได้เลยคับ

🔹 ขั้นตอนการเติมเกมแบบ UID

  • 1. แจ้งแพ็คที่ต้องการเติม
  • 2. แจ้ง UID (บางเกมต้องใช้ Server ด้วย)
  • 3. โอนเงินพร้อมแจ้งสลิป รอคิวเติม
  • 4. เติมเสร็จแล้วเข้าเกมเช็คของได้เลยคับ

💰 เติมขั้นต่ำ 30 บาท

⭐ สะสมครบ 10 แต้ม แลกรับส่วนลดหรือแพ็คเล็กสุดได้คับ

🔹 ขั้นตอนเติมเกมแบบ Mail - Pass

  • 1. แจ้งแพ็คที่ต้องการเติมและโอนเงิน
  • 2. แจ้งชื่อในเกม UID Server พร้อม Mail - Pass
  • 3. รอกดยืนยันเข้าเกมให้ทางร้าน (หากเลยเวลาอาจข้ามคิว)
  • 4. ระหว่างเติมห้ามเข้าไอดีเด็ดขาด
  • 5. เติมเสร็จแล้วเข้าเช็คของได้เลยคับ

รายละเอียดการเติม

แบบ Mail - Pass

  • ใช้รหัสเข้าดิสคอร์ด
  • ราคาถูกกว่า
  • หากเปิดยืนยันตัวตน 2 ชั้น แนะนำให้ปิดก่อน หรือแจ้งรหัส 3-4 ตัว

แบบ Gift

  • ไม่ต้องใช้รหัส
  • กดรับจากลิ้งได้เลย
  • ราคาสูงกว่าแบบ Mail-Pass

🔹 ขั้นตอนเติมไนโตรแบบ Mail - Pass

  • 1. แจ้งรายการที่ต้องการเติม (หากเป็นกรอบรูปส่งรูปให้ร้านได้เลย)
  • 2. แจ้ง Mail - Pass (ถ้ามี 2 ชั้นแจ้งรหัส 3-4 ตัว)
  • 3. รอกดยืนยัน
  • 4. รอทางร้านเติม หากติดยืนยันคำสั่งซื้อจะมีแจ้ง
  • 5. เติมเสร็จแล้วเข้าไอดีเช็คได้เลยคับ
